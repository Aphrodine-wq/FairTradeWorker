## Quick Reference: The Three Routes

| | Route A: Vertical SaaS | Route B: Marketplace | Route C: Data & API |
|---|---|---|---|
| **One-liner** | Best contractor tools in construction | Match homeowners with verified contractors | Sell ConstructionAI estimation intelligence |
| **Cold start** | None | High risk | None |
| **First revenue** | 1-2 months | 3-6 months | Already (MHP) |
| **Revenue ceiling** | $10M+/yr | $100M+/yr | $5-20M/yr |
| **SBIR fit** | Strong | Moderate | Very strong |
| **Recommended order** | 1st | 3rd | 2nd |

**Route D: Mississippi Blitz** — The accelerated version. All three routes compressed into 12 months with $67K investment (covered by SBIR). 1,000 contractors, statewide ads, field reps, HBA partnerships, brand ambassadors. $27,750 MRR by month 12. See [route-d-blitz.md](08-strategy/route-d-blitz.md).

**Recommended sequence: A → C → B (or D if funded).** Tools first (build supply), monetize data (fund growth), marketplace last (compound network effects). If marketplace never works, A + C are still viable standalone businesses.
